Webserver

Ein Einfacher HTTP Server.
Der auch das schreiben von text-basierenden Dateien,
wie ".html, .json, .md, .csv, .txt, ...usw" unterstützt.

Dateien können in den Ordner "data" oder "_build" geschrieben und gelesen werden.

Zum Beispiel: 

        fetch("/data/test.json", {
            method: "POST", 
            body:'{"name": "Max Muster", "alter": 42}'
        });
